# KuyHide Helper — LSPosed/Xposed Module
### by KuyangID | [Telegram](https://t.me/KuyangID)

All-in-one Xposed module to hide debugging, ADB, and accessibility from apps.

## Features
- ✅ Hide USB Debugging (ADB) status from apps
- ✅ Hide Developer Options status  
- ✅ Hide Accessibility services from apps
- ✅ Hide debugger connection (isDebuggerConnected)
- ✅ Spoof system properties (ro.debuggable, ro.secure, etc)

## How to Get the APK
1. Go to **Actions** tab in this repo
2. Click the latest **Build Xposed Module** workflow
3. Download **KuyHide-Helper** artifact
4. Extract → you get the APK!

## Install
1. Install the APK on your phone
2. Open **LSPosed Manager** → Modules → Enable **KuyHide Helper**
3. Select scope: **System Framework** + target apps
4. Reboot

## Requirements
- Magisk + LSPosed installed
- Android 8.0+
